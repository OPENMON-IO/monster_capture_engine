# Contributing to OpenMon MCE

Thank you for your interest in the Monster Capture Engine!

## How to Get Involved

MCE is a commercial engine available via license from [OpenMon.IO](https://www.openmon.io). This public repository hosts the **API reference** and **code samples** — not the full engine source.

### Want to use MCE in your game?

Visit [openmon.io](https://www.openmon.io) to join the waitlist for early access.

### Found a bug?

Open an [Issue](https://github.com/OPENMON-IO/monster_capture_engine/issues) with:
- MCE version
- Unity version
- Steps to reproduce
- Expected vs actual behavior

### Have a feature request?

Open an [Issue](https://github.com/OPENMON-IO/monster_capture_engine/issues) with the `enhancement` label.

### Want to chat?

Join our [Discord](https://discord.gg/zszqu7sPuY) for engine dev discussion, game design, and help building monster-catching RPGs.
